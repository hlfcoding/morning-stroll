# Morning Stroll

## Development

```bash
$ npm install

# to install
$ grunt install

# to read some docs
$ grunt docs

# to start developing
$ grunt
```

### Sample ST2 Project File

```json
{
  "folders":
  [
    {
      "path": "morning-stroll",
      "file_exclude_patterns":
      [
        "docs/*",
        "lib/*",
        "release/*",
        "tests/specs/*"
      ],
      "folder_exclude_patterns":
      [
        ".grunt",
        "node_modules"
      ]
    }
  ]
}

```
