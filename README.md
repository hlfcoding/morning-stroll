# Morning Stroll

## Development [![devDependency Status](https://img.shields.io/david/dev/hlfcoding/morning-stroll.svg)](https://david-dm.org/hlfcoding/morning-stroll#info=devDependencies)

```bash
$ npm install

# to install
$ grunt install

# to read some docs
$ grunt docs

# to run unit tests
$ grunt test

# to start developing
$ grunt
```
