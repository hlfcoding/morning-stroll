Morning Stroll: About

The game was originally a class project hacked together with Flixel over a couple weeks. This version is the Phaser port of that initial version. It is also currently in beta testing, so please 'fiddle' with the settings and provide feedback in the issue tracker. Check back sometime for the final release. We have mobile aspirations!

For education purposes, the entire project is open source. You can even read the annotated source here. Your support in any $ amount is welcome! Donate here.
